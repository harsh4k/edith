# EDITH

Minimal system terminal dashboard with real-time monitoring.

---

## Overview

EDITH is a lightweight terminal-based system interface designed for clarity and control.  
It provides real-time insights into system performance while maintaining a clean, distraction-free UI.

---

## Features

- Real-time CPU, RAM, Disk usage
- Cache memory tracking
- Temperature monitoring (CPU/GPU, if available)
- Optional GPU usage support
- Live monitoring mode
- Clean command-driven interface
- Warm-colored terminal UI
